Method - видит всё, поэтому в метод добавлять саму структуру, не нужно!!

Если в методе совпадает название принимаемого параметра с параметром в структуре, то
self.параметр 								- обозначается параметр из структуры (за пределами метода который)
Если параметры не совпадают, то self - писать не нужно!

self 										- указывает на тот объект, который вызывается (и с ним идёт обработка)

Вызвал класс - селф - класс,
вызвал экземпляр класса - селф будет экземпляр класса
вызвал енум - будет енумом


************************************        # Method
mutating func 								- mutating, используется когда нужно менять параметры в самой структуре!


************************************        # примеры
struct Point {
    var x: Int
    var y: Int
    
    mutating func movePoint(x: Int, byY: Int) {
        self.x += x 						// тут совпали название переменных структуры и метода (поэтому self - указывает на внешний параметр)
    }
}

var point1 = Point.init(x: 1, y: 1)

point1.movePoint(x: 5, byY: 1)
